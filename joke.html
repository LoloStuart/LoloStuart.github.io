About this project

The assignment was to use an API for anything we wanted. I chose a "dad joke" API and created a simple computer graphic to act as the speaker of the jokes. To get a little experience with CSS animations, I learned how to make the eyes blink by transforming a couple green squares.

Why is the conputer named SHAWN? It's a reference to one of my bootcamp instructors, who likes to tell dad jokes.
